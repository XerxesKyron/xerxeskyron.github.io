ClariThink: A Platform for Inattentive-type ADHD's Reading Problems and Focus Retention Using Structured Machine-Learning Models

    ClariThink is an assistive tool designed to address the reading difficulties of individuals with Inattentive ADHD (ADHD-I). The system integrates three core features: text summarization using the T5 model, text-to-image generation with the Stable Diffusion model, and question generation powered by a T5-based model. These components work together to help individuals with ADHD-I retain focus and enhance comprehension during reading tasks. The project aimed to provide personalized interventions for ADHD-I, a condition that particularly impacts sustained attention. ClariThink's implementation has been preliminarily evaluated by a psychologist, with feedback emphasizing the need for user interface improvements like color scheme and font adjustments to optimize user experience.



Challenges:

  1. Model Performance:
    • The T5 text summarization model achieved a BLEU score of 0.2195, outperforming alternatives like BART and GPT-2 but still faced limitations, such as a 512-word input cap.
    • Stable Diffusion's text-to-image conversion achieved a CLIP score of 29.827 but required further refinement for better alignment with text content.
    • The T5-based question generation model showed promising semantic coherence with a score of 0.662 but required improvement in grammatical accuracy, indicated by a METEOR score of 0.2119.
  2. System Features:
    • The lack of user management and database functionality limited the systems ability to scale for broader use.
    • User feedback highlighted the need for better interface elements, such as accessible fonts and colors.
  3. Target Audience Testing:
    • Initial evaluations were conducted by a psychologist, but testing with children diagnosed with ADHD-I, educators, and mental health professionals is essential to validate and refine the system.

Learnings:

  1. Technology Integration:
    • The project demonstrated the potential of T5 and Stable Diffusion models in creating personalized educational tools for neurodivergent individuals.
    • Performance metrics like BLEU, CLIP, and METEOR scores provided critical insights into the strengths and weaknesses of the implemented models.
  2. User-Centered Design:
    • Feedback from professionals underscored the importance of user-friendly design elements, which are crucial for tools targeting individuals with cognitive challenges.
  3. Future Development:
    • Expanding the text limit of the summarization model and incorporating user management features are vital steps for scalability.
